# MERN Task Manager

## Tech Stack
- MongoDB
- Express.js
- React.js (Vite)
- Node.js

## Backend Setup
cd backend
npm install
npm start

Create a .env file:
PORT=5000
MONGO_URI=your_mongo_url
JWT_SECRET=your_secret

## Frontend Setup
cd frontend
npm install
npm run dev

## Features
- Admin & Employee Login
- Task Assignment
- Task Status Update
- Task History
