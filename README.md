# MERN Task Manager


## Tech Stack
- MongoDB
- Express.js
- React.js (Vite)
- Node.js

## Backend Setup
cd backend
npm install
npm start

## Frontend Setup
cd frontend
npm install
npm run dev

## Features
- Admin & Employee Login
- Task Assignment
- Task Status Update
- Task History
